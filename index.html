LEDee is an MVR

A Minimum Viable Robot is an exercise in building a robot with the highest amount of functionality with the least amount of parts.

As of this moment, the LEDee hardware is composed of an arduino duemilanove, an arduino prototyping shield, an LED, and some jumper cables.

The software is currently being written around a set of protocols.

The LEDee Protocols.

Protocols are being developed to standarize all of the basic functionality for LEDee. The first protocal is called the LEDee Communication Protocol. It defines how LEDee will communicate with other LEDees through an internet connection.

There will be various protocols, though an official listing is yet not defined.

Project goals

  • Define a set of simple protocols for communicating state between robots and machines
  • Introduce people to simple robotics and AI.
